# 数据准备简要步骤

## 步骤 1：下载并解压 SemanticKITTI

1. 打开官网：<http://www.semantic-kitti.org/dataset.html#download>
2. 下载数据集并解压，使最终目录结构为：

```
DistillationDPO/
└── datasets/
    └── SemanticKITTI/
        └── dataset/
            └── sequences/
                ├── 00/   (含 velodyne/、labels/ 等)
                ├── 01/
                └── ...
```

确保每个序列（如 `00`）下存在 `velodyne/`、`labels/`、`calib.txt`、`poses.txt`。

---

## 步骤 2：生成地面真值地图

在项目根目录执行：

```bash
python map_from_scans.py --path datasets/SemanticKITTI/dataset/sequences/
```

脚本会为各序列生成 `map_clean.npy`。若仅用 CPU，可加 `--cpu`：

```bash
python map_from_scans.py --path datasets/SemanticKITTI/dataset/sequences/ --cpu
```

---

## 步骤 3：下载教师模型权重

1. 打开：<https://drive.google.com/drive/folders/1z7Iq6nPDZXtASUDP8R8sqhUAvVfRqKQH?usp=sharing>
2. 下载 **lidiff_ddpo_refined.ckpt**
3. 放到项目下：`checkpoints/lidiff_ddpo_refined.ckpt`

若目录不存在，先创建：`mkdir -p checkpoints`

---

## 完成后启动训练

```bash
python trains/DistillationDPO.py --SemanticKITTI_path datasets/SemanticKITTI --pre_trained_diff_path checkpoints/lidiff_ddpo_refined.ckpt
```
